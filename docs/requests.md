# Как работать с запросами в приложении

Так как это Next js приложение, стоит помнить что тут есть компоненты двух видов: серверные и клиентские. Запросы осуществляются на серверной стороне для оптимизации производительности.

Серверный компонент начинает запрос и возвращает Promise с данными.
Этот промис мы созраняем в контекст, из которого достаем этот промис и данные из него в любом компоненте внутри контекста.
